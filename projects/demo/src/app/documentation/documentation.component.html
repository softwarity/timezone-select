Documentation

Installation

npm install @softwarity/timezone-select

Usage

import { TimezoneSelectComponent } from '@softwarity/timezone-select';
import { MatFormFieldModule } from '@angular/material/form-field';

@Component({
  imports: [TimezoneSelectComponent, MatFormFieldModule, ReactiveFormsModule],
  template: `
    <mat-form-field>
      <mat-label>Timezone</mat-label>
      <timezone-select [formControl]="control" />
    </mat-form-field>
  `
})
export class MyComponent {
  control = new FormControl('Europe/Paris');
}

API

The component implements ControlValueAccessor and MatFormFieldControl, making it fully compatible with Angular forms and Material form fields.

Input Type Default Description
placeholder string '' Placeholder text (used by mat-form-field)
required boolean false Whether the field is required
disabled boolean false Disables the component
showDetectButton boolean false Shows a button to detect and set the browser's timezone

Outputs

Output Type Description
valueChange string | null Emitted when the selected timezone changes

Features

  • UTC Offset Navigation - Browse timezones by offset using chevron buttons (infinite loop)
  • Browser Timezone Detection - Optional button to auto-detect and set browser timezone
  • Browser Timezone Indicator - Visual marker on the user's browser timezone in dropdown
  • Native Timezone Data - Uses Intl.supportedValuesOf('timeZone') for accurate timezone list
  • Material Integration - Works seamlessly inside mat-form-field
  • Reactive Forms - Full support for FormControl and ControlValueAccessor
  • Dark Mode - Automatically adapts to light/dark color scheme

Theming (Material 3)

The component provides a SCSS mixin to integrate with your Material 3 theme:

@use '@angular/material' as mat;
@use '@softwarity/timezone-select/timezone-select-theme' as timezone-select;

html {
  @include mat.theme((
    color: (primary: mat.$violet-palette)
  ));

  // Apply timezone-select theme
  @include timezone-select.theme();
}
